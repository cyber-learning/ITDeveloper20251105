
Gli elementi block hanno queste particolarità:
- width 100%
- sono impilati in verticale (uno sotto l'altro)

Gli elementi inline hanno queste particolarità:
- hanno una width strettamente necessaria a contenere il proprio contenuto TESTUALE
- sono impilati in orizzontale (uno di fianco l'altro)


Gli elementi inline NON rispettano il box model... per evitare questo problema possiamo trasformare gli elementi inline in inline-block tramite display



Proprietà display al valore:
- none
- block
- inline
- inline-block
- flex

E' presente una proprietà che non vi è stata spiegata cioè backdrop-filter.

La proprietà backdrop-filter permette di applicare degli effetti grafici, come il blur (sfocato), al background dell'elemento.
