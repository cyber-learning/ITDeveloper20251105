WWW è un insieme di 3 tecnologie:
- URI -> URL Location -> individuare un documento all'interno della rete
- HTTP 
- HTML linguaggio di formattazione per i documenti



Architettura Client-Server
Due programmi di cui uno assume il ruolo di client e l'altro il ruolo di server.
SOLO il programma client può avviare la comunicazione con il programma server (effettuando una richiesta)
Il programma server può solo rispondere alla richiesta del client.

Il programma server NON può mai avviare la comunicazione con il client!!!



URI: uniform resource identifier è un modo generico (astratto) utile per identificare in maniera univoca una risorsa nella rete
URL: uniform resource locator (caso particolare di URI) posso individuare in maniera univoca una risorsa nella rete conoscendo la sua collocazione nella rete (dove si trova)
