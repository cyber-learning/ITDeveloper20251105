Una variabile è un'area di memoria in RAM con un nome il cui valore è alterabile






in JS i valori possono essere di tipo differente (tipi di dato)
- tipi di dato primitivi
- tipi di dato riferimento


Tra i tipi di dato primitivi ci sono:
- number
- string
- boolean
- undefined significa valore non definito


una espressione è una combinazione di valori tramite operatori
che restituiscono un nuovo valore come risultato



operatori aritmetici tra numbers (operatori binari a 2 valori alla volta):
+
-
*
/
% resto della divisione (detto anche modulo)
** elevamento a potenza

operatori di assegnamento composti:
+=
-=
*=
/=
%=

operatore di incremento (di 1)
x++;


operatore di decremento (di 1)
x--;
