viewport: la porzione di documento HTML visibile agli occhi dell'utente


unità di misura:
- assolute
- relative (a qualcos'altro)


unità di misura assolute:
- px, mm, cm

unità di misura relative (utili per layout):
- % rispetto alla width dell'elemento parent. Eccezione alla regola: se utilizzate il % per la height dell'elemento allora % si riferisce alla height dell'elemento parent e non alla width dell'elemento parent
- vw rispetto alla viewport width
- vh rispetto alla viewport height







unità di misura relative (utili per font - vediamo prossima volta):
- em
- rem
